# GestureInstrument

# Setup Instructions

Download and Install Orion Leap Service SDK
Hyperlink:_>>>>>

Plug in Leap Motion controller

Download and Install GestureInstrument Plugin 

The VST folder includes the LeapC service and .vst which are both needed for usage.

Copy vst along with LeapC to your VST folder location.

Open DAW of your choice and enjoy!!

Standalone

Included in the package the the standalone plugin which can be used without the need of a DAW.
